## Notification

You can set up notifications for the following QA.tech events:

- Test run failed
- Test run succeeded
- Weekly report

## Platforms

We currently support sending Slack and email notifications. If you'd like support for another notification platform, [please reach out to us](LINK-TO-CONTACT-PAGE)

### Slack

Configure Slack notifcations at [Settings => Connections](https://app.qa.tech/dashboard/current-project/settings/organization/connections)

### Email

Email notifications are enabled by default for all users.
